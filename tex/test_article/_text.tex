\subsubsection*{Non-breaking spaces}
Tildes don't appear in mathmode. Here is an fbox containing lots of them:
\par
\fbox{$A~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~B$}
\par
This is useful to protect mathmode against VLE rewriters.


\subsection{Accents}
Delimited
\begin{itemize}
\item Not spaced: \'{e}\`{e}\^{w}\"{o}.
\item Spaced: \'{e} \`{e} \^{w} \"{o}.
\end{itemize}

Undelimited
\begin{itemize}
\item Not spaced: \'e\`e\^w\"o.
\item Spaced: \'e \`e \^w \"o.
\end{itemize}

Some words ...
\begin{itemize}
\item Cram\'{e}r-Rao.
\item Y m\^{o}r coch.
\end{itemize}

\subsection{Escaped characters}
\begin{itemize}
\item Not spaced: \&\%\$.
\item Spaced: \& \% \$ .
\end{itemize}	

We need to be careful with \$ characters!
